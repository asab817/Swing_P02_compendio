# Swing_P02_compendio

Aplicación de escritorio desarrollada en Java **Swing** como práctica de la asignatura, que recopila y demuestra distintos componentes de interfaz gráfica (botones, menús, cuadros de diálogo, menús desplegables, tablas, etc.).  

## Descripción del proyecto

Este proyecto tiene como objetivo practicar el desarrollo de interfaces gráficas con Java Swing, organizando en una sola aplicación un compendio de ejemplos y funcionalidades habituales en aplicaciones de escritorio.  
Incluye gestión de eventos, validación básica, uso de diferentes contenedores y componentes gráficos, así como una estructura de código preparada para generar javadoc.

## Contenido de la publicación

En el repositorio de GitHub se incluyen los siguientes elementos:

- `desarrollo/`  
  Contiene el proyecto completo de la práctica **Swing_P02_compendio** con su estructura de directorios de código fuente y recursos.

- `ejecutable/`  
  Incluye el fichero `.jar` ejecutable de la aplicación, listo para ser ejecutado sin necesidad de abrir el IDE.

- `documentacion_tecnica/`  
  Contiene la documentación técnica generada con **javadoc**, incluyendo `index.html` y el resto de ficheros necesarios para la consulta de la API del proyecto.

- `documentos/`  
  Incluye el documento sobre la usabilidad de la aplicación (análisis de la interfaz, facilidad de uso, mejoras propuestas, etc.).

- `README.md`  
  Archivo de documentación principal del repositorio, redactado en **Markdown**.

## Desarrollo del proyecto

- Lenguaje: Java.  
- Paradigma: Programación Orientada a Objetos (POO).  
- Biblioteca gráfica: Swing.  
- IDE utilizado: Eclipse.  
- Principales características:
  - Uso de ventanas y diálogos modales y no modales.
  - Implementación de menús y barras de herramientas.
  - Gestión de eventos de botones y otros componentes.
  - Organización del código en paquetes y clases bien diferenciadas.

## Despliegue

### Windows

1. Asegúrate de tener instalada una versión de **Java** (JRE/JDK) compatible con el proyecto.  
2. Descarga o clona el repositorio en tu equipo.  
3. Accede a la carpeta `ejecutable/` donde se encuentra el archivo `.jar` del proyecto.

#### Ejecución con doble clic

- En la mayoría de configuraciones de Windows, basta con hacer doble clic sobre el archivo  
  `ArchivoJar.jar` para iniciar la aplicación.

#### Ejecución desde línea de comandos

1. Abre una ventana de **Símbolo del sistema (CMD)** o **PowerShell**.  
2. Navega hasta la carpeta `ejecutable/`. Por ejemplo:

   ```bash
   cd ruta\al\repositorio\ejecutable

3. Ejecuta el siguiente comando:
   
  java -jar NombreDelArchivo.jar
